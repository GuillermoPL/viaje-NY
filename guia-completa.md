# 🗽 Bienvenidos a Nueva York

Nueva York no es una ciudad para verla.

Es una ciudad para vivirla.

Durante estos días vais a caminar muchísimo, descubriréis barrios completamente diferentes entre sí, comeréis probablemente la mejor pizza de vuestra vida y entenderéis por qué tanta gente vuelve una y otra vez.

El objetivo de esta guía no es visitar el mayor número posible de monumentos.

El objetivo es **disfrutar Nueva York**.
