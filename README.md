**Sí, absolutamente.**

Aunque nunca hayáis visto un partido de béisbol, asistir a un encuentro de los Yankees es una de las experiencias más auténticas que podéis vivir en Nueva York.

📅 Viernes 21 de agosto

🕖 19:05
